🧠 AI & Algorithms Toolkit

فارسی (Persian)

🚧 Work in Progress – This repository is being actively developed. More projects and documentation will be added soon.

A collection of four interactive Python/Tkinter applications covering graph search, robot localization, N‑Queens, and 8‑Puzzle.
All applications are fully bilingual (English & Persian) and use only the Python standard library — zero external dependencies.

📦 Applications

#ApplicationDescription
1Graph Search AlgorithmsVisualises tree/graph traversal using DFS, BFS, ID, UCS, A*, plus stochastic methods (Hill Climbing, SA, Beam, GA).
2Robot Localization + PathfindingSimulates a robot that localises itself using predict‑update cycles and navigates with multiple pathfinding algorithms.
3N-Queens SolverSolves the N‑Queens problem with deterministic and stochastic algorithms; displays all/unique solutions, supports N≥4.
48-Puzzle GamePlayable 8‑puzzle with A* solver using Manhattan heuristic; supports custom boards.

🚀 How to Run

  1. Clone the repository (or download the ZIP): 
    git clone https://github.com/alirezaghavami/ai-toolkit.git
cd ai-toolkit
  2. Navigate to the desired application folder: 
    cd graphsearch      # or robot_localization / nqueens / 8puzzle
  3. Run with Python (3.7+ required): 
    python graphsearch.py
    No additional packages are needed — everything is in the standard library.

🔨 Building Standalone Executables (PyInstaller)

You can create a single executable (.exe on Windows, binary on Linux/macOS) using PyInstaller:

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." graphsearch.py
  • On Linux/macOS replace ; with : in --add-data.
  • If a .spec file is provided, use pyinstaller app.spec.
  • The output will be inside the dist/ folder.

🌐 Language Switching

Click the FA | EN button on the top bar to switch between English and Persian.
Translation files (.po/.mo) reside in the locale/ folder. To compile updated .po files, use:

msgfmt locale/fa/LC_MESSAGES/graphsearch.po -o locale/fa/LC_MESSAGES/graphsearch.mo
