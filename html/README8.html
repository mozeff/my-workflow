🤖 مکانیابی ربات + الگوریتم‌های مسیریابی

English

شبیه‌سازی رباتی که با چرخهٔ predict‑update مکان خود را پیدا می‌کند و با ۱۱ الگوریتم (تصادفی، حریصانه، BFS، DFS، A*، الگوریتم ژنتیک و ...) حرکت می‌کند.
شامل ویرایشگر نقشه، تعیین مقصد و رابط کاربری کاملاً دو زبانه (انگلیسی و فارسی) – بدون هیچ وابستگی خارجی.

📥 دانلود نسخهٔ اجرایی (ویندوز)

فایل .exe آماده از آخرین نسخهٔ منتشرشده:
🔽 دانلود robot_localization.exe

✨ امکانات

  • مکانیابی احتمالاتی با چرخهٔ predict‑update
  • ۷ نقشهٔ پیش‌فرض + ویرایشگر نقشهٔ دلخواه
  • ۱۱ الگوریتم حرکت (تصادفی، تصادفی جهت‌دار، حریصانه، SA، پرتو محلی، BFS، DFS, IDDFS, UCS, A*, الگوریتم ژنتیک)
  • تعیین مقصد و هدایت خودکار ربات
  • نمایش دایره‌های باور (مکان‌های احتمالی ربات)
  • کنترل گام‌به‌گام دستی یا پخش خودکار
  • رابط کاربری تاریک، دو زبانه (انگلیسی / فارسی)

🚀 اجرا از روی کد منبع

  1. دریافت مخزن: 
    git clone https://github.com/alirezaghavami/robot-localization.git
cd robot-localization
  2. اجرای اسکریپت (نیاز به پایتون ۳.۷ یا بالاتر): 
    python robot_localization.py

هیچ بستهٔ اضافی نیاز نیست – همه چیز در کتابخانهٔ استاندارد پایتون موجود است.

🔨 ساخت فایل اجرایی با PyInstaller

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." robot_localization.py
  • در لینوکس/macOS جداکنندهٔ --add-data را : بگذارید.
  • خروجی در پوشهٔ dist/ قرار می‌گیرد.
