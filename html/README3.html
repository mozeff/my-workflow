🧩 8‑Puzzle Solver & Game

فارسی (Persian)

An interactive 8‑puzzle game with a built‑in A* solver (Manhattan heuristic).
Play manually, enter your own puzzles, or let the AI solve them step‑by‑step.
Fully bilingual (English & Persian) — zero external dependencies.

📥 Download Executable (Windows)

You can grab the ready‑to‑run .exe file from the latest release:
🔽 Download 8puzzle.exe

✨ Features

  • Classic 8‑puzzle (3×3 sliding tiles)
  • A* solver with Manhattan distance heuristic
  • Manual play with clickable tiles
  • Custom board input (check solvability)
  • Step‑by‑step AI animation (stop at any time)
  • Track user and AI move counts
  • Dark mode GUI with bilingual UI (English / Persian)

🚀 How to Run from Source

  1. Clone the repository: 
    git clone https://github.com/alirezaghavami/8puzzle.git
cd 8puzzle
  2. Run the script (Python 3.7+ required): 
    python 8puzzle.py

No additional packages needed — the entire application uses only the Python standard library.

🔨 Build Your Own Executable (PyInstaller)

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." 8puzzle.py
  • On Linux/macOS replace ; with : in --add-data.
  • The output will be inside the dist/ folder.

🌐 Language Switching

Press the FA | EN button in the top bar to toggle between English and Persian.
