🌳 الگوریتم‌های جست‌وجو در گراف (با متاهیوریستیک‌ها)

English

اجرای بصری الگوریتم‌های کلاسیک جست‌وجو روی گراف‌های دلخواه.
شامل روش‌های قطعی (DFS, BFS, ID, UCS, A*, SAP) و متاهیوریستیک‌های تصادفی (Hill Climbing, Simulated Annealing, Beam Search, Genetic Algorithm).
کاملاً دو زبانه (انگلیسی و فارسی) – بدون هیچ وابستگی خارجی.

📥 دانلود نسخهٔ اجرایی (ویندوز)

فایل .exe آماده از آخرین نسخهٔ منتشرشده:
🔽 دانلود graphsearch.exe

✨ امکانات

  • ورود گراف‌های دلخواه (یال‌ها با وزن) و هیوریستیک‌های اختیاری
  • ۱۰ الگوریتم: DFS, BFS, ID, UCS, A*, SAP, Hill Climbing, Simulated Annealing, Beam Search, Genetic Algorithm
  • نمایش گراف/درخت به‌صورت تعاملی با قابلیت اسکرول
  • توضیح فرمول هر الگوریتم
  • رابط کاربری تاریک، دو زبانه (انگلیسی / فارسی)

🚀 اجرا از روی کد منبع

  1. دریافت مخزن: 
    git clone https://github.com/alirezaghavami/graph-search.git
cd graph-search
  2. اجرای اسکریپت (نیاز به پایتون ۳.۷ یا بالاتر): 
    python graphsearch.py

هیچ بستهٔ اضافی نیاز نیست – همه چیز در کتابخانهٔ استاندارد پایتون موجود است.

🔨 ساخت فایل اجرایی با PyInstaller

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." graphsearch.py
  • در لینوکس/macOS جداکنندهٔ --add-data را : بگذارید.
  • خروجی در پوشهٔ dist/ قرار می‌گیرد.
