🌳 Graph Search Algorithms (with Metaheuristics)

فارسی (Persian)

Visualise and run classic graph‑search algorithms on custom graphs.
Includes deterministic methods (DFS, BFS, ID, UCS, A*, SAP) and stochastic metaheuristics (Hill Climbing, Simulated Annealing, Beam Search, Genetic Algorithm).
Fully bilingual (English & Persian) — zero external dependencies.

📥 Download Executable (Windows)

Ready‑to‑run .exe from the latest release:
🔽 Download graphsearch.exe

✨ Features

  • Input custom graphs (edges with weights) and optional heuristics
  • 10 algorithms: DFS, BFS, ID, UCS, A*, SAP, Hill Climbing, Simulated Annealing, Beam Search, Genetic Algorithm
  • Interactive tree/graph drawing with drag‑scroll support
  • Formula explanation for each algorithm
  • Dark mode GUI, bilingual (English / Persian)

🚀 How to Run from Source

  1. Clone the repository: 
    git clone https://github.com/alirezaghavami/graph-search.git
cd graph-search
  2. Run the script (Python 3.7+ required): 
    python graphsearch.py

No additional packages needed — everything is in the Python standard library.

🔨 Build Your Own Executable (PyInstaller)

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." graphsearch.py
  • On Linux/macOS replace ; with : in --add-data.
  • The output will be inside the dist/ folder.

🌐 Language Switching

Press the FA | EN button in the top bar to toggle between English and Persian.
