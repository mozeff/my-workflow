♛ حل‌کنندهٔ N‑Queen با چندین الگوریتم

English

یک برنامهٔ تعاملی برای حل مسئلهٔ N‑Queen با پشتیبانی از ۱۰ الگوریتم (DFS, Backtracking, BFS, IDDFS, UCS, A*, Hill Climbing, Simulated Annealing, Local Beam Search, Genetic Algorithm).
نمایش تمام جواب‌ها، جواب‌های یکتا یا یک جواب؛ صفحهٔ شطرنج متحرک، حالت تاریک و کاملاً دو زبانه (انگلیسی و فارسی) – بدون هیچ وابستگی خارجی.

📥 دانلود نسخهٔ اجرایی (ویندوز)

فایل .exe آماده از آخرین نسخهٔ منتشرشده:
🔽 دانلود nqueens.exe

✨ امکانات

  • حل N‑Queen برای N ≥ ۴ (تا ۱۴‑۱۵ به‌صورت عملی)
  • ۱۰ الگوریتم – قطعی و تصادفی
  • پیدا کردن یک جواب یا همهٔ جواب‌ها
  • فیلتر جواب‌های یکتا (حذف تقارن‌ها)
  • صفحهٔ شطرنج متحرک با امکان رنگ‌های متمایز برای وزیرها
  • معیارهای کارایی (زمان، نرخ خطا برای الگوریتم‌های تصادفی)
  • پنل توضیح الگوریتم
  • رابط کاربری تاریک، دو زبانه (انگلیسی / فارسی)
  • بزرگ‌نمایی صفحه و اسلایدشو خودکار

🚀 اجرا از روی کد منبع

  1. دریافت مخزن: 
    git clone https://github.com/alirezaghavami/nqueens.git
cd nqueens
  2. اجرای اسکریپت (نیاز به پایتون ۳.۷ یا بالاتر): 
    python nqueens.py

هیچ بستهٔ اضافی نیاز نیست – همه چیز در کتابخانهٔ استاندارد پایتون موجود است.

🔨 ساخت فایل اجرایی با PyInstaller

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." nqueens.py
  • در لینوکس/macOS جداکنندهٔ --add-data را : بگذارید.
  • خروجی در پوشهٔ dist/ قرار می‌گیرد.
