🧩 پازل ۸ – بازی و حل‌کنندهٔ خودکار

English

یک بازی تعاملی پازل ۸ همراه با حل‌کنندهٔ A* (فاصلهٔ منهتن).
می‌توانید به‌صورت دستی بازی کنید، پازل دلخواه وارد کنید یا اجازه دهید هوش مصنوعی گام‌به‌گام حل کند.
کاملاً دو زبانه (انگلیسی و فارسی) – بدون هیچ وابستگی خارجی.

📥 دانلود نسخهٔ اجرایی (ویندوز)

فایل اجرایی .exe آماده را می‌توانید از آخرین نسخهٔ منتشرشده دریافت کنید:
🔽 دانلود 8puzzle.exe

✨ امکانات

  • پازل ۸ کلاسیک (صفحهٔ ۳×۳)
  • حل‌کنندهٔ A* با فاصلهٔ منهتن
  • بازی دستی با کلیک روی کاشی‌ها
  • وارد کردن صفحهٔ دلخواه (بررسی قابلیت حل)
  • انیمیشن گام‌به‌گام حل توسط هوش مصنوعی (قابل توقف)
  • شمارش تعداد حرکات شما و هوش مصنوعی
  • رابط کاربری تیره (Dark mode) و دو زبانه

🚀 اجرا از روی کد منبع

  1. دریافت مخزن: 
    git clone https://github.com/alirezaghavami/8puzzle.git
cd 8puzzle
  2. اجرای اسکریپت (نیاز به پایتون ۳.۷ یا بالاتر): 
    python 8puzzle.py

هیچ بستهٔ اضافی نیاز نیست – همه چیز در کتابخانهٔ استاندارد پایتون موجود است.

🔨 ساخت فایل اجرایی با PyInstaller

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." 8puzzle.py
  • در لینوکس/macOS جداکنندهٔ --add-data را : بگذارید.
  • خروجی در پوشهٔ dist/ قرار می‌گیرد.
