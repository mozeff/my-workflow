An interactive N‑Queens solver supporting 10 algorithms (DFS, Backtracking, BFS, IDDFS, UCS, A*, Hill Climbing, Simulated Annealing, Local Beam Search, Genetic Algorithm).
View all solutions, unique solutions, or a single result. Animated board, dark mode, and fully bilingual (English & Persian) — zero external dependencies.

📥 Download Executable (Windows)

Ready‑to‑run .exe from the latest release:
🔽 Download nqueens.exe

✨ Features

  • Solve N‑Queens for any N ≥ 4 (up to 14‑15 for practical use)
  • 10 algorithms – deterministic and stochastic
  • Find a single solution or all solutions
  • Filter to unique solutions (symmetry‑reduced)
  • Animated chessboard with distinct queen colours option
  • Performance metrics (time, error rate for stochastic methods)
  • Algorithm explanation panel
  • Dark mode GUI, bilingual (English / Persian)
  • Zoomable board, auto‑play slideshow

🚀 How to Run from Source

  1. Clone the repository: 
    git clone https://github.com/alirezaghavami/nqueens.git
cd nqueens
  2. Run the script (Python 3.7+ required): 
    python nqueens.py

No extra packages required — everything is in the Python standard library.

🔨 Build Your Own Executable (PyInstaller)

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." nqueens.py
  • On Linux/macOS replace ; with : in --add-data.
  • The output will be inside the dist/ folder.

🌐 Language Switching

Press the FA | EN button in the top bar to toggle between English and Persian.
