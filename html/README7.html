🤖 Robot Localization + Pathfinding Algorithms

فارسی (Persian)

Simulates a robot that localises itself using predict‑update cycles and navigates with 11 algorithms (random, greedy, BFS, DFS, A*, genetic algorithm, etc.).
Includes built‑in map editor, destination setting, and fully bilingual UI (English & Persian) — zero external dependencies.

📥 Download Executable (Windows)

Ready‑to‑run .exe from the latest release:
🔽 Download robot_localization.exe

✨ Features

  • Probabilistic localization using predict‑update cycle
  • 7 pre‑built maps + custom map editor
  • 11 movement algorithms (random, biased random, greedy, SA, beam, BFS, DFS, IDDFS, UCS, A*, Genetic Algorithm)
  • Set a destination and let the robot navigate automatically
  • Animated belief circles showing possible robot positions
  • Manual step‑by‑step control or auto‑play
  • Dark mode GUI, bilingual (English / Persian)

🚀 How to Run from Source

  1. Clone the repository: 
    git clone https://github.com/alirezaghavami/robot-localization.git
cd robot-localization
  2. Run the script (Python 3.7+ required): 
    python robot_localization.py

No additional packages needed — everything is in the Python standard library.

🔨 Build Your Own Executable (PyInstaller)

pip install pyinstaller
pyinstaller --onefile --windowed --add-data "locale;locale" --add-data "logo.png;." robot_localization.py
  • On Linux/macOS replace ; with : in --add-data.
  • The output will be inside the dist/ folder.

🌐 Language Switching

Press the FA | EN button in the top bar to toggle between English and Persian.
